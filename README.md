# 🌦 Weather App (TypeScript + REST API)

A simple and attractive Weather Application built using TypeScript and OpenWeather REST API.

## 🚀 Features

- Search weather by city name
- Displays temperature (°C)
- Shows weather description and icon
- Error handling for invalid city
- Loading message while fetching data
- Animated Particles.js background
- Responsive and clean UI

## 🛠 Tech Stack

- TypeScript
- JavaScript
- OpenWeather REST API
- Particles.js
- HTML & CSS

## 📁 Project Structure

weather-app-ts/
│
├── src/app.ts
├── dist/app.js

├── style.css
└── tsconfig.json

## ▶ How to Run

1. Install dependencies  
   npm install

2. Compile TypeScript  
   npx tsc

3. Run using Live Server  
   live-server
